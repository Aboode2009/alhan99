# Music Player

مشروع Vite/React/TypeScript مع دعم تشغيل الأغاني، التنزيل للأوفلاين، واستيراد ملفات صوتية محلية.

## التشغيل محلياً
```bash
npm install
npm run dev
```

## البناء
```bash
npm run build
```

## أندرويد (Capacitor)
```bash
npm run build
npx cap sync android
cd android
./gradlew assembleDebug
```
